# discrete LQR controller based on C++
* This project gives an example, which can be modified by users.    
* Every block during option start and option end should be modified by users according 
  to the actual situation, where contains necessary parameters and printing options.   
* Need to pre install dependency `Eigen`:    
  `sudo apt-get install libeigen3-dev`      
  `sudo ln -s /usr/include/eigen3/Eigen /usr/include/Eigen`    
* Provide simple parameter type checking function.
